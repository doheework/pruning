The ‘pruning’ typeface
expresses the activity
of trimming the tree.
This font is made of
the variable typeface.
So, you can use this
on any Adobe suite,
like other normal fonts.
The difference is that
you can customise
the cutting stage
to change the shape!
(Of course, this function is
available on Adobe Suite!)

The typeface is still developing.
The font will release with a completed set
and function in 2023!
